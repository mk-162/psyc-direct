# Our Commitment

We are committed to providing a website and clinic experience that is accessible to the widest possible audience, regardless of technology or ability.

| Principle | Our Approach |
|-----------|--------------|
| **Inclusive Design** | Considering accessibility from the start |
| **Continuous Improvement** | Regular review and enhancement |
| **User Feedback** | Listening to accessibility needs |
| **Compliance** | Meeting WCAG 2.1 AA standards |

# Website Accessibility

## Standards Compliance

We aim to conform to WCAG 2.1 Level AA standards.

| Feature | Implementation |
|---------|----------------|
| **Keyboard Navigation** | Full site accessible without mouse |
| **Screen Reader Compatible** | ARIA labels and semantic HTML |
| **Text Resizing** | Compatible with browser zoom |
| **Colour Contrast** | Sufficient contrast ratios |
| **Alt Text** | Descriptive text for images |

## Accessibility Features

| Feature | How to Use |
|---------|------------|
| **Text Size** | Use browser zoom (Ctrl/Cmd + +/-) |
| **High Contrast** | Browser or OS settings |
| **Screen Reader** | Compatible with major screen readers |
| **Keyboard Shortcuts** | Tab to navigate, Enter to select |
